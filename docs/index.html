// 03 · Erste Schritte nach dem Flash

Einrichtung

  1. ESP32 startet einen WLAN Access Point namens LD2410S-Setup (Passwort: ld2410s-setup)
  2. Mit diesem WLAN verbinden, dann im Browser http://192.168.4.1 öffnen
  3. Im Webportal WLAN- und MQTT-Zugangsdaten eintragen und speichern
  4. Home Assistant erkennt alle Entitäten automatisch per MQTT Auto Discovery
